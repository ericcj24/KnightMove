KnightMove
This program try to calculate the possible moves for a knight to traverse on
a alphabet+numeric board:
A B C D E
F G H I J
K L M N O
  1 2 3

the program take the number of moves, n, as input, and calculate all possible
moves to form a path that contains 2 or less vowel.

there is a program argument available for running the program: verbose:true
if set to true, the program will print out all path in terminal, but please note
this will significantly sacrifice the performance of the program.
